# Blog Management System

## About the Project

This is a simple Blog Management System developed using PHP and MySQL. The project allows users to register, log in, and manage blog posts through Create, Read, Update, and Delete (CRUD) operations.

## Features

* User Registration
* User Login
* User Logout
* Add New Blog Posts
* View Blog Posts
* Edit Blog Posts
* Delete Blog Posts
* Responsive User Interface

## Technologies Used

* HTML
* CSS
* Bootstrap 5
* PHP
* MySQL
* Git & GitHub

## Project Structure

* register.php – User Registration
* login.php – User Login
* dashboard.php – View All Posts
* add_post.php – Add New Post
* edit_post.php – Update Post
* delete_post.php – Delete Post
* logout.php – Logout User
* db.php – Database Connection

## How to Run

1. Start Apache and MySQL in XAMPP.
2. Create a database named `blog`.
3. Import/Create the required tables.
4. Place the project inside `xampp/htdocs`.
5. Open:

http://localhost/blog_app
